# Walkthrough Final — Proyecto Esencia de Luz

## Resumen del Proyecto
Hemos transformado el sitio web **Esencia de Luz** en una plataforma de e-commerce completa y profesional, con un sistema de gestión de inventario local, carrito interactivo y documentación técnica de alto nivel.

## 🚀 Nuevas Funcionalidades Implementadas

### 1. 🛒 Carrito de Compras de Próxima Generación
- **Panel Deslizable (Drawer):** Un carrito moderno que no interrumpe la navegación.
- **Cálculo de Envío en Tiempo Real:** Selector de rutas (Local, Nacional, Recogida en tienda) que suma el costo al total instantáneamente.
- **Sincronización de Inventario:** El carrito verifica el stock antes de permitir añadir productos.

### 2. 📦 Gestión de Inventario Inteligente
- **Bloqueo de Agotados:** Los productos sin stock se marcan como "Agotado" y sus botones se desactivan automáticamente.
- **Panel de Administración:** Una herramienta dedicada ([admin-inventario.html](file:///c:/Users/flora/OneDrive/Escritorio/ESENCIA-DE-LUZ/ESENCIA-DE-LUZ/admin-inventario.html)) para que el dueño del negocio ajuste las unidades disponibles de forma manual.
- **Deducción Automática:** Al completar un pedido por WhatsApp, el sistema resta las unidades del inventario local.

### 3. ✨ Experiencia del Usuario (UX) Elevada
- **Banner Promocional:** Una barra superior llamativa para anuncios de "Envío Gratis".
- **Social Proof:** Sección de testimonios de clientes con calificaciones de 5 estrellas.
- **Galería de Instagram:** Una cuadrícula visual que conecta la marca con sus redes sociales.
- **Preguntas Frecuentes (FAQ):** Una página completa con sistema de acordeón para resolver dudas comunes.

### 4. 📄 Documentación Técnica (SRS)
- Se entregó un documento de **Especificación de Requisitos de Software** profesional ([SRS_Esencia_de_Luz_v1.0.docx.md](file:///c:/Users/flora/OneDrive/Escritorio/ESENCIA-DE-LUZ/ESENCIA-DE-LUZ/SRS_Esencia_de_Luz_v1.0.docx.md)) que sirve como manual técnico y base para futuros desarrollos.
